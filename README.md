## Anibate

Anibateはアニメ好きな人が自分の感想を語り合ったり、視聴作品を記録できたりするプラットフォームです。
豊富なアニメ作品に基づいた感想投稿、簡単な視聴記録をすることができます。


## 使用技術
### バックエンド
- PHP 7.3.28
- Laravel 6.20.27
- [Annict API](https://docs.annict.com/docs/ja/)

### フロントエンド
- HTML
- CSS
- Bootstrap4
- Vue.js 2.6.12
